# SoftSole — Web Development & Software Studio

A production-ready agency website for **SoftSole**, a web development and software solutions studio. Built with **Next.js 16**, **React 19**, and **Tailwind CSS v4**.

## Design

- **Light hero** on off-white with bold geometric typography (**Plus Jakarta Sans**)
- **Neon lime** (`#c6ff00`) and **purple** (`#6b4eff`) accents
- **Dark sections** on near-black with charcoal cards and lime headings

## Features

- **5 page types**: Home (`/`), Services (`/services`), Work (`/work` + case studies), Blog (`/blog` + posts), Contact (`/contact`)
- **Dark mode** via `next-themes`
- **Scroll animations**, bento grid layouts, Cal.com booking CTAs
- **SEO**: metadata, JSON-LD, sitemap, robots.txt

## Getting started

```bash
pnpm install
pnpm dev      # http://localhost:3000
```

```bash
pnpm build
pnpm start
pnpm lint
```
